Quick little mention about javascript scope

What is 'scope', and how does it work in JavaScript?

Scope refers to the accessibility of different variables declared in a document. For example, we might have a variable called 'score' that tracks how many points you've scored in a match of insert name of your preferred sport. Scope refers to what other parts of the code are capable of interacting with this 'score' variable. In Javascript, the rules for this are very simple. Either the variable is 'global', accesible to any other code; Or the variable is 'local', only 'seen' by code within a function. Another way to put it is: either your score sheet is right on top of the dresser where it's visible to everyone (global scope); Or, it's in one of the drawers (a function) where it will only be visible when you open that particular drawer (local scope).
